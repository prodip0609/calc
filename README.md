## Calc
A simple CLI calculator built in GO using Cobra(https://github.com/spf13/cobra)

### Run the app
``` bash
make run
```

### Usage

Get command help
``` bash
calc --help
```

Add two numbers
``` bash
calc add 5 6
```

Subtract two numbers
``` bash
calc subtract 5 6
```

Multiply two numbers
``` bash
calc multiply 5 6
```

Divide two numbers
``` bash
calc divide 5 6
```
